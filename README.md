# PhoenixOS v1

PhoenixOS v1 is a preview refactor of the [SJTU-IPADS/phoenixos](https://github.com/SJTU-IPADS/phoenixos) project. The refactor focuses on making PhoenixOS easier to use and adds checkpoint-and-restore support for vLLM; see the [vLLM guide and examples](tests/apps/vllm/README.md).

- Remoting component and `config.toml` docs: [docs/remoting.md](docs/remoting.md)
- Environment docs: [environment/README.md](environment/README.md)
